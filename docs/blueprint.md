# **App Name**: Shanti Palace Management Suite

## Core Features:

- Guest-Facing Website: Public website with a focus on aesthetics. Guests can browse room categories, check availability, and submit booking requests.
- Admin Panel: Internal tool for managing bookings, rooms, guests, housekeeping, and maintenance. Role-based access control for different staff members.
- Booking Management: Allows staff to view, create, and manage bookings from both walk-ins and website submissions, displaying source of the booking (Walk-in vs. Website).
- Room and Category Management: Allows admin and front desk staff to manage room details, categories, pricing, and availability. Changes reflect automatically on the public website.
- Housekeeping Module: Housekeeping staff can access and update room cleaning status from mobile devices or tablets.
- Enquiry Management: Allows admin and front desk staff to manage the website enquiries submitted from public web site
- Booking details auto-generation: Automatically generate a booking based on Hotels.com prices. An LLM acts as a tool to provide best matching prices and availability to present to the user in the booking form, easing user data input requirements.

## Style Guidelines:

- Primary color: Gold (#B8924A) for a luxurious and elegant feel, in line with the 'Shanti Palace' brand.
- Background color: Pale Gold (#F5ECD9) for a warm and inviting ambiance, creating a peaceful atmosphere.
- Accent color: Deep Brown (#3D2B1F) for text and sidebar, providing a strong contrast for readability and a grounded feeling.
- Headings font: 'Cormorant Garamond' (serif) for an elegant and classic look in titles and headings. Note: currently only Google Fonts are supported.
- Body font: 'DM Sans' (sans-serif) for clear and highly readable text in body content and UI elements. Note: currently only Google Fonts are supported.
- Use clear, recognizable icons alongside text labels on buttons and in room feature lists, for universal comprehension. Include icons in these areas:
- All CTAs should have easily understood accompanying icons
- Primary nav links must use accompanying icons (home, rooms, about, contact) 
- Housekeeping and maintenance boards on Admin panel
- Room categories (Standard, Deluxe, etc.)
- Room ameneties, i.e. mini bar
- Employ a clean and spacious layout with generous padding between elements to ensure clarity and ease of use, especially on mobile devices. Make forms wizard steps take up the full screen, especially on mobile.
- Subtle transitions and loading animations should be implemented to enhance user experience. Animations include subtle transitions to new pages and while loading data into tables and wizards, in addition to clear indicators for successfully submitted data.
